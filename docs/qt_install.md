## 【Ubuntu版 qt6インストール手順】　　　　　　
#### ［[ＴＯＰ](./)**｜**[目次](./#目次)**｜**[ドキュメント](./#ドキュメント-1)]

**Ubuntuにqt6をインストールする手順です。**
**端末から以下の手順でコマンドを実行します。**

### １．Ubuntuを最新の状態に更新

```
    sudo apt update && sudo apt upgrade -y
```
　　　　　　

### ２．qt6をインストール

```
    sudo apt install qt6-base-dev
```
　　　　　　　

**※もしかしたら不足のモジュールを追加してインストールが必要かもしれません。**
